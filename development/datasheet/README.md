# Datasheet Directory

This directory is intended for NTC thermistor datasheets.

## Note

NTC thermistors are generic components with specifications that vary by manufacturer and part number. The driver supports multiple NTC types including:

- **NTCG163JFT103FT1S** (TDK)
- **NTCG164JF103FT1S** (TDK)
- **NTCG163JF103FT1S** (TDK)
- Custom NTC types with configurable parameters

## Finding Datasheets

Datasheets for specific NTC thermistor parts can be found on manufacturer websites:

- **TDK**: https://product.tdk.com/
- **Murata**: https://www.murata.com/
- **Vishay**: https://www.vishay.com/
