Detailed Description

Hardware-agnostic NTC thermistor driver for temperature measurement.

This header provides a comprehensive interface for temperature measurement using NTC (Negative Temperature Coefficient) thermistors. It supports various NTC types and provides both lookup table and mathematical conversion methods.
