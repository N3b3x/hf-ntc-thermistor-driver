Detailed Description

template<typename Derived>
class NTC::NtcAdcInterface< Derived >

CRTP-based template interface for ADC operations.

This template class provides a hardware-agnostic interface for ADC operations using the CRTP pattern. Platform-specific implementations should inherit from this template with themselves as the template parameter.

Benefits of CRTP:

  • Compile-time polymorphism (no virtual function overhead)
  • Static dispatch instead of dynamic dispatch
  • Better optimization opportunities for the compiler

Example usage:

class Esp32C6Adc : public NTC::NtcAdcInterface<Esp32C6Adc> {
public:
bool IsInitialized() const { ... }
bool EnsureInitialized() { ... }
// ... implement other methods
};
