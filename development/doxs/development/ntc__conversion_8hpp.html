Detailed Description

Mathematical conversion functions for NTC thermistor temperature calculation.

This header provides mathematical conversion methods for calculating temperature from NTC thermistor resistance values using the Steinhart-Hart equation and beta parameter approximation.

All conversion functions are in the NTC namespace and can be used independently of the NtcThermistor class for standalone calculations.
