Detailed Description

Lookup table implementation for NTC thermistor temperature conversion.

This header provides lookup table functionality for fast temperature conversion from resistance values. It includes pre-calculated tables for common NTC types.

Lookup tables provide a balance between speed and accuracy, making them ideal for real-time applications where computational resources are limited.

Note
Lookup tables use binary search and linear interpolation for efficient temperature lookups. Accuracy depends on table resolution.
