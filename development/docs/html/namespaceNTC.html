bool NTC::ConvertResistanceToTemperatureBeta ( float resistance_ohms,
float resistance_at_25c,
float beta_value,
float * temperature_celsius )
noexcept

Convert resistance to temperature using beta parameter.

Uses the beta parameter equation to calculate temperature from resistance. This is a simplified model that assumes a linear relationship in the temperature-resistance curve around the reference temperature.

Note
The beta parameter method is faster but less accurate than Steinhart-Hart, especially at temperature extremes.
Parameters
resistance_ohmsThermistor resistance (ohms)
resistance_at_25cResistance at 25°C (ohms)
beta_valueBeta value (K), typically 3000-4000K for NTC thermistors
temperature_celsiusPointer to store temperature (°C)
Returns
true if successful, false otherwise
Warning
Input resistance must be positive and non-zero. Beta value should be within valid range (1000-5000K).

bool NTC::ConvertResistanceToTemperatureSteinhartHart ( float resistance_ohms,
float coeff_a,
float coeff_b,
float coeff_c,

noexcept

Convert resistance to temperature using Steinhart-Hart equation.

Uses the Steinhart-Hart equation for high-accuracy temperature calculation. This method provides better accuracy across a wider temperature range compared to the beta parameter method.

The Steinhart-Hart equation: 1/T = A + B*ln(R) + C*ln(R)³

Note
This method is more computationally expensive but provides superior accuracy, especially at temperature extremes.
Parameters
resistance_ohmsThermistor resistance (ohms)
aSteinhart-Hart coefficient A (typically ~1.129e-3)
bSteinhart-Hart coefficient B (typically ~2.341e-4)
cSteinhart-Hart coefficient C (typically ~8.775e-8)
temperature_celsiusPointer to store temperature (°C)
Returns
true if successful, false otherwise
See also
ValidateSteinhartHartCoefficients() to validate coefficients
