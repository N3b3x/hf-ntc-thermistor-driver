Detailed Description

Abstract base class for ADC interface.

This class provides a hardware-agnostic interface for ADC operations. Platform-specific implementations should inherit from this class and implement all pure virtual methods.

Example usage:

class Esp32C6Adc : public NTC::BaseAdc {
public:
bool IsInitialized() const override { ... }
bool EnsureInitialized() override { ... }
// ... implement other methods
};
